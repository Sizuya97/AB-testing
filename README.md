# А/B-тест и аналитическая рекомендация
Данный проект состоит из ноутбука ab_test.ipynb с кодом на языке Python и файл с выборкой ab_test.csv. В данной работе был проведён A/B-тест и дана аналитическая рекомендация с помощью библиотек pandas/numpy/scipy/matplotlib/seaborn.

Для того чтобы просмотреть работу, необходимо скачать файлы ab_test.ipynb, затем открыть в Jupyter Notebooks, установив дистрибутив Anaconda, или пройти по ссылке https://github.com/Sizuya97/parsing/blob/main/ab_test.ipynb.
